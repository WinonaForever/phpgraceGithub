**PHPgrace框架**

轻量级框架，PHPgrace轻快的实力派!

重写内核 2.0版本
**开发环境:**

```
解析引擎: Apache/nginx  
PHP版本: PHP > 5.3  (使用命名空间)
数 据库:  mysql / sqlServer

```

**注意**
```

支持根目录下的静态文件访问，不支持application下静态目录访问
/statics/images/1.jpg  仅支持访问根目录下的图片文件

不支持 ./statics/imgs/1.jpg  
也不支持  admin/statics/imgs/1.jpg  这样应用模块访问静态文件  访问不了
```
